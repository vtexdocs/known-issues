## Sumário

Na página de edição de "Meu endereço" da seção "Minha conta", mesmo que o endereço esteja preenchido, a mensagem em vermelho "Campo obrigatório" permanece exibida até que os dados sejam salvos.

## Simulação

Acesse a página "Meu endereço", selecione a opção de edição e, após preencher o endereço, a mensagem de aviso continuará exibida até que os dados sejam salvos.
