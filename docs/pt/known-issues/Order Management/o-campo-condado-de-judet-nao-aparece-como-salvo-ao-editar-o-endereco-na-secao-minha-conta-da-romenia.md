## Sumário

Para lojas da Romênia, na página _**Minha conta**_, há um campo chamado "Judet" (condado, em português), que não aparece como salvo ao editar o endereço.

## Simulação

- Ao verificar o endereço em "Minha conta", é possível ver o endereço salvo anteriormente.
- Então, é possível ver que o comportamento desse campo não é o mesmo dos outros campos.
- Clique em “Editar endereço” e o endereço anterior ainda estará nos campos de endereço, exceto no campo “Judet”.

## Workaround

Não há solução alternativa disponível para isso na própria página “Minha conta”, mas talvez seja possível usar nossas APIs de dados mestre para atualizar ou criar esses endereços.
