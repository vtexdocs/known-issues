## Sumário

 Uma das etapas para criar um produto é inserir um slug para representar a URL da página do produto. Se a URL contiver qualquer caractere especial, ela gerará uma URL inválida e poderá causar problemas ao tentar acessar a página do produto. Em outras palavras, o catálogo não remove caracteres especiais ao configurar o link de texto, como © ℗ ® ™
