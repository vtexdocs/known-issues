## Sumário

O link “Por que preencher a Categoria Global do VTEX?” na configuração de categorias só funciona quando aberto em uma nova aba. Ao tentar abri-lo apenas clicando no link, o usuário é redirecionado para uma página de erro.

 ![](https://vtexhelp.zendesk.com/attachments/token/Hhun8gyXHxWgk1qnUHWreWIBo/?name=image.png)

## Simulação

1. Acesse uma categoria existente ou crie uma nova categoria
2. Tente clicar neste link “Por que preencher a Categoria Global VTEX?”
3. Uma página de erro será exibida
4. Tente abrir em uma nova guia ou janela
5. O link correto será exibido

## Workaround

Abra em uma nova guia.
