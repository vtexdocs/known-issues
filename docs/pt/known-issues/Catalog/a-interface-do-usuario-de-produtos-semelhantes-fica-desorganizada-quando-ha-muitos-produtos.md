## Sumário

Ao abrir a ficha de um produto com mais de 6 produtos semelhantes, a visualização da interface do usuário apresenta falhas
e, em vez de exibir os produtos em formato de lista, passa a exibi-los em formato escalonado

## Simulação

Adicione 10 produtos semelhantes a um produto

## Workaround

- Recomendamos usar a API para gerenciar os produtos semelhantes
- Tente minimizar o zoom na página para que ela se ajuste melhor; isso funcionará em alguns casos
