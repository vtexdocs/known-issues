## Sumário

Ao criar um SKU no Admin, é necessário registrar as dimensões para envio e as dimensões reais. No entanto, quando você preenche um dos campos de texto e clica (ou usa a tecla Tab) para preencher outro, esse campo é desmarcado e é necessário clicar no campo de texto novamente para poder inserir/editar o valor.

## Simulação

- Vá para o Catálogo e crie um novo SKU em um produto
- Preencha o primeiro campo “Peso de envio”
- Clique no campo seguinte ou use a tecla Tab
- Observe que, após um breve momento, o campo fica desmarcado, não sendo possível editá-lo

## Workaround

Clique novamente no campo de texto
