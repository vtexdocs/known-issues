## Sumário

A movimentação de categorias na árvore de categorias, por exemplo, a transferência de um departamento para outra categoria, não atualiza o ID do departamento nos produtos.

## Simulação

- Mova uma categoria para outra posição, dentro de outra categoria
- Verifique se o departamento dos produtos na categoria movida não é atualizado com as informações corretas
