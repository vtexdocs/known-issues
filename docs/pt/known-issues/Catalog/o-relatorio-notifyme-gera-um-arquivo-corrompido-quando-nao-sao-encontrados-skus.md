## Sumário

Quando o relatório “Notify Me” não contém SKUs, tentar exportá-lo gera um arquivo corrompido no Excel

## Simulação

- Verifique se o relatório “Notify Me” está vazio
- Se sim, clique em “Exportar para o Excel”
- Baixe o arquivo e verifique se é possível abri-lo

## Workaround

Abra o arquivo em outro software ou preencha o relatório para poder exportá-lo
