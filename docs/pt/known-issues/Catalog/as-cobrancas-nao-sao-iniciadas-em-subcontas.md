## Sumário

Ao tentar abrir uma coleção no novo módulo de coleções de uma subconta, é exibida a mensagem de erro “Desculpe, ocorreu um erro desconhecido”.

## Simulação

1. Acesse uma subconta;
2. Vá para o novo módulo de coleções;
3. Tente abrir qualquer coleção;
4. Verifique se ocorreu um erro.
5. Acesse a mesma coleção, mas na conta principal; ela será exibida normalmente.

## Workaround

Abra a coleção na conta principal e faça as alterações ou use o módulo de coleções antigo em _**Storefront -> Layout**_
