## Sumário

Quando um SKU é enviado por um vendedor e aprovado no módulo “SKUs recebidos”, as imagens do novo SKU no catálogo do marketplace não seguem a ordem enviada pelo vendedor.

No entanto, esse comportamento nem sempre ocorre.

## Simulação

1. O vendedor enviou um SKU para o marketplace;
2. O SKU é aprovado no módulo “SKUs recebidos”;
3. O SKU é criado no catálogo com as imagens fora de ordem.

## Workaround

Não há solução alternativa.
