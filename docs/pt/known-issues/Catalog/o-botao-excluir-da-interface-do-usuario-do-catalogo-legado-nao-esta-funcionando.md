## Sumário

O botão para excluir produtos do catálogo não está funcionando; ele simplesmente não responde aos cliques do usuário

 ![](https://vtexhelp.zendesk.com/attachments/token/ZR7ogOzhfLN3yC61V9RzTMRlZ/?name=inline-652081372.png)

Essa ação pode funcionar se o produto tiver sido criado recentemente e não houver SKUs, imagens ou qualquer outro tipo de dado associado a ele.

## Simulação

- Acesse a página principal do catálogo;
- Tente excluir um produto: nenhuma ação será acionada

## Workaround

Não há como excluir apenas um único produto, apenas toda a base. Para obter o mesmo resultado que uma exclusão, tente substituir esse produto pelos dados que você já usaria para um novo produto de qualquer maneira.
