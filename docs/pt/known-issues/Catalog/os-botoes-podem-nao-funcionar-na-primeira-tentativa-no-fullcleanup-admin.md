## Sumário

No painel de administração do Fullcleanup, espera-se que os botões funcionem na primeira tentativa. No entanto, para executar a ação, pode ser necessário clicar no botão várias vezes.

## Simulação

1. Abra o Fullcleanup admin - `/Admin/Site/FullCleanUp.aspx  `
2. Clique no botão **Reindexar banco de dados**
3. Pode não ocorrer nada

## Workaround

Tente clicar no botão mais vezes
