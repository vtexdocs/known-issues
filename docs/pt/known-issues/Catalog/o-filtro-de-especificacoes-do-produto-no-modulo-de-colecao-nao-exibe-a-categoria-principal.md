## Sumário

No módulo de coleção, ao usar os filtros para adicionar produtos, o filtro "Especificações do produto" não exibe os campos registrados na categoria raiz

## Simulação

- Em uma coleção, na opção "Mais", selecione o filtro "Especificações do produto"
- Um novo campo será exibido: "Digite uma categoria"
- Tente localizar a categoria raiz

## Workaround

Use os outros filtros do módulo, importe os produtos por meio de uma planilha ou adicione os produtos individualmente na interface do usuário
