## Sumário

Ao utilizar o cartão-presente de fidelidade, os usuários podem ter uma experiência negativa ao selecionar a opção “Usar apenas parte do valor”, pois não há um botão “Adicionar” após preencher o campo “Valor a ser usado” (o valor é aceito automaticamente); assim, se o valor não for digitado com rapidez suficiente, ele será atualizado apenas com os primeiros dígitos, obrigando o usuário a repetir o procedimento. Essa experiência é ainda pior no site para dispositivos móveis.

## Simulação

Após criar um cartão-presente de fidelidade para um e-mail, ele aparecerá no checkout e você poderá prosseguir com o cenário descrito acima.
