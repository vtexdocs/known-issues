## Sumário

Os componentes de um kit têm o mesmo preço quando uma promoção de brinde é configurada e a entrega é feita por um vendedor regular.

## Simulação

- Registre um kit;
- Defina uma promoção de brinde;
- Adicione um item ao carrinho utilizando um vendedor regular;
- Finalize o pedido; o preço dos componentes será o mesmo.
