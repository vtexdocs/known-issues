## Sumário

>ℹ️ Este problema conhecido foi traduzido automaticamente do inglês.


No módulo Master Data, o mecanismo de pesquisa do administrador redireciona para o URL errado.

## Simulação

Admin > Pesquisar > Dados mestre >
