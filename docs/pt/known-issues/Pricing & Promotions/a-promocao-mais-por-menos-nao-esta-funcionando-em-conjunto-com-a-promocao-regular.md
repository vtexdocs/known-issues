## Sumário

O desconto final no carrinho de compras não está correto quando uma promoção “Mais por menos” é aplicada em conjunto com uma promoção regular. O desconto é maior do que o esperado.

## Simulação

1. Crie uma promoção 3x2;
2. Crie uma promoção regular, digamos, com 15% de desconto, que se aplique ao mesmo produto da promoção anterior;
3. Marque a opção para permitir o acúmulo na segunda promoção;
4. Verifique se o desconto no carrinho é maior do que o esperado.

## Workaround

Não há solução alternativa para este caso.
