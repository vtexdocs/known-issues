## Sumário

Ao deslizar a imagem de um produto na visualização rápida no celular, quando há um layout de slider no fundo, ambos os sliders avançam para a próxima imagem, em vez de apenas a imagem na visualização rápida

## Simulação

- Abra uma visualização rápida no celular com um layout de slider no fundo
- Tente deslizar a imagem do produto

O comportamento esperado é que não apenas a imagem do produto na visualização rápida mude, mas também os produtos no layout do slider/prateleira
