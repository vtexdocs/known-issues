## Sumário

Ao visualizar o PLP no modo de lista, as tags dos itens desaparecem.

## Simulação

Crie tags para seus itens e faça as alterações necessárias nas configurações.
Habilite a visualização do PLP tanto no modo de grade quanto no modo de lista.
No estilo de grade, as tags aparecerão como deveriam.
No estilo de lista, as tags estão faltando.
