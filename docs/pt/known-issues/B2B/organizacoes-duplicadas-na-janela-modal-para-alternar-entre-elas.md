## Sumário

Quando um usuário tem várias organizações cadastradas e usa a janela modal para alternar entre elas, algumas organizações aparecem duplicadas na lista.

## Simulação

- Configure várias organizações para o mesmo e-mail;
- Faça login e abra a janela modal para alternar entre as organizações;
- Algumas organizações aparecerão duplicadas na lista.
