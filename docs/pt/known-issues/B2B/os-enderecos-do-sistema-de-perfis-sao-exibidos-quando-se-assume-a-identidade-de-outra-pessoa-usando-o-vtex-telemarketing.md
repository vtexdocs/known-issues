## Sumário

Quando o comprador possui endereços registrados no Sistema de Perfil e é representado por meio do VTEX Telemarketing, os endereços do Sistema de Perfil e das Organizações são exibidos

## Simulação

- Represente um comprador que tenha endereços registrados no Sistema de Perfil;
- Monte um carrinho e vá para Frete;
- Selecione um endereço; o componente será atualizado e exibirá todos os endereços do Sistema de Perfis e das Organizações.

## Workaround

- Use a função de representação de Organizações B2B.
