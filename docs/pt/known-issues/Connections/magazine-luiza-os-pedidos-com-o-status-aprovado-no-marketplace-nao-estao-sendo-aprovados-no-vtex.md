## Sumário

Alguns pedidos com status APROVADO no marketplace não estão sendo aprovados no VTEX

## Simulação

Esses pedidos ficam estagnados no status “Aguardando autorização para envio” aqui no VTEX, mesmo estando aprovados no marketplace

- Pedido aprovado no marketplace;
- O status do pedido não foi atualizado para aprovado no OMS (VTEX);
- O log não mostra nenhum erro no fluxo;
- A ponte não mostra nenhum erro;
