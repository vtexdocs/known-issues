## Sumário

Esperava-se que, ao marcar a caixa de seleção de pré-venda, a coleção fosse preenchida com produtos cuja data de lançamento tivesse sido definida para uma data futura.
No entanto, isso não está ocorrendo, pois a coleção não é preenchida, permanecendo sempre vazia.

## Simulação

- Defina uma data de lançamento futura para alguns produtos
- Crie uma coleção
- Acesse a administração da coleção no CMS legado
- Configure a coleção para produtos em pré-venda
- Verifique que a coleção nunca é preenchida

## Workaround

Gerencie manualmente os produtos na coleção
