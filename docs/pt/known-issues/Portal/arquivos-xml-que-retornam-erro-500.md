## Sumário

Em alguns casos, o XML pode retornar um erro 500; isso ocorre principalmente com arquivos XML de grande porte.

## Simulação

Tente carregar um arquivo XML de grande porte.

## Workaround

- Tentar recarregar o XML geralmente resolve o problema;
- Use mais de um arquivo XML.
