## Sumário

Quando um filtro booleano é selecionado, é exibida a seguinte mensagem:
`Ocorreu um erro inesperado. Tente novamente. Se o problema persistir, entre em contato com o suporte.`
 ![](https://vtexhelp.zendesk.com/attachments/token/HAwA6Uf51x49bxGLiziKpdQnj/?name=image.png)

## Simulação

- Acesse o Master Data CRM (por exemplo, https://my-account-here.vtexcrm.com.br);
- Acesse um formulário de aplicativo com a opção de filtro booleano:
 ![](https://vtexhelp.zendesk.com/attachments/token/tyRYoZP61rVzhVBx3A1rHbGgf/?name=image.png)

- Selecione a opção 0 ou 1;
- Clique no botão "_Filtro_";
- Uma mensagem pop-up será exibida com o erro "`Ocorreu um erro inesperado. Tente novamente. Se o problema persistir, entre em contato com o suporte.":
 ![](https://vtexhelp.zendesk.com/attachments/token/HAwA6Uf51x49bxGLiziKpdQnj/?name=image.png)
