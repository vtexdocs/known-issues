## Sumário

 Alguns aplicativos criados usando o Master Data build 2.x podem apresentar erros de tempo de execução imediatamente após serem instalados em uma conta onde não foram instalados anteriormente (instalação nova). A associação automática entre a versão do aplicativo instalado e sua definição de dados (esquema) falha. Como resultado, o sistema não consegue localizar a estrutura necessária para salvar ou ler informações, retornando o erro “Nenhum esquema associado foi encontrado”.
