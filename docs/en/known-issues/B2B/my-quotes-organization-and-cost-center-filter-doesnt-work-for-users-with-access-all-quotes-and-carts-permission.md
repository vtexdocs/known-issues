## Summary

When logging in to the My Quotes page with a user that has "Access All Quotes and Carts" permission and opening the "Organization and Cost Center" Filter, the user will not be able to search for an organization. Typing on the search bar has no effect and the autocomplete returns no results.

## Simulation

- In a store using the B2B Suite and the `b2b-quotes` app, set the "Access All Quotes and Carts" permission to a user
- Log in with this user
- Go to My Account -> My Quotes
- Select the "Organization and Cost Center" Filter
- Type in the search bar; you will see nothing will be typed, and the autocomplete will return no options to be selected
