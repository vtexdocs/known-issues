## Summary

Filters in the "Received SKUs" section stop working when the user navigates beyond the first page of results. After applying a filter and moving to a subsequent page, the filter criteria are no longer respected, causing all records to be displayed regardless of the active filter.

## Simulation

1. Navigate to the Received SKU section.
2. Apply any in the search box, like SKU ID.
3. Verify that the filter is applied correctly on the first page of results.
4. Navigate to the second page (or any page beyond the first).
5. Observe that the filter is no longer applied and no records are displayed.

## Workaround

Alternatively, when the search is needed go back to first page and apply the filter again.
