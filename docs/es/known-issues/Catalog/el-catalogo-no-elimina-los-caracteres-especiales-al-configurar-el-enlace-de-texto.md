## Sumario

 Uno de los pasos para crear un producto es insertar una etiqueta que represente la URL de la página del producto. Si la URL contiene algún carácter especial, generará una URL no válida y podría causar problemas al intentar acceder a la página del producto. En otras palabras, el catálogo no elimina los caracteres especiales al configurar el enlace de texto, como © ℗ ® ™
