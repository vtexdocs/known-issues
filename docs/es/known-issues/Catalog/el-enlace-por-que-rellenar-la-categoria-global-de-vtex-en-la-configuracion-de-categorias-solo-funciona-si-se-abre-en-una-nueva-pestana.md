## Sumario

El enlace «¿Por qué rellenar la categoría global de VTEX?» en la configuración de categorías solo funciona si se abre en una nueva pestaña. Si se intenta abrirlo simplemente haciendo clic en el enlace, se redirige a una página de error.

 ![](https://vtexhelp.zendesk.com/attachments/token/Hhun8gyXHxWgk1qnUHWreWIBo/?name=image.png)

## Simulación

1. Ve a una categoría existente o crea una nueva categoría
2. Intenta hacer clic en este enlace «¿Por qué rellenar la categoría global de VTEX?»
3. Aparecerá una página de error
4. Intenta abrirlo en una nueva pestaña o ventana
5. Aparecerá el enlace correcto

## Workaround

Ábrelo en una nueva pestaña.
