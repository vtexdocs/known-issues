## Sumario

El mensaje en portugués «Não é possível desactivar este produto porque possui um ou mais skus ativos.» carece de traducciones adecuadas en la administración del catálogo.

## Simulación

Si se utiliza cualquier idioma distinto del portugués, en la interfaz de usuario `/produtoform.aspx`, al pasar el cursor por encima de la etiqueta «Producto activo» aparecerá este mensaje:


 No es posible activar este producto porque no tiene ningún SKU activo.
