## Sumario

Al crear un servicio, el comerciante puede elegir si va a ser un campo obligatorio en la página de pago o no.
Sin embargo, actualmente esta funcionalidad no funciona como se esperaba. Aunque el campo obligatorio esté marcado como «verdadero», el producto se puede comprar sin añadirlo en la página de pago.

## Simulación

1. Crea un servicio como obligatorio;
2. Asócialo a un SKU;
3. Comprueba que, en la caja, el servicio se puede añadir o eliminar sin problemas.
