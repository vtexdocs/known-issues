## Sumario

Actualmente, la pestaña «elementos personalizados» de las configuraciones del CMS heredado no funciona correctamente.

En la mayoría de los casos, configurar un elemento personalizado de este tipo no surtirá efecto, por lo que se recomienda configurarlo directamente en el código HTML del diseño.

Esto solo ocurre en las tiendas con plantillas heredadas.

## Simulación

1 - Ve al CMS heredado y selecciona la pestaña de elementos personalizados
2 - Sube un elemento personalizado del tipo «banner»
3 - Haz referencia a este banner en tu HTML
4 - El banner configurado no se mostrará correctamente

## Workaround

Configura los banners utilizando el tipo de diseño HTML y/o las subplantillas asociadas a él.

O

Migrar al marco de tiendas de VTEX
