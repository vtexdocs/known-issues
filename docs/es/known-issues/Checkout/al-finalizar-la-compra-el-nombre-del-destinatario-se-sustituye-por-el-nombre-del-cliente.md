## Sumario

El nombre del destinatario se sustituye por el nombre del cliente tras volver a cargar la página de pago.

## Simulación

- Crea un nuevo carrito
- Continúa como nuevo cliente con el nombre «A»
- Introduce una dirección, rellenando el campo del destinatario con el nombre «B»
- Ve al paso de pago
- Recarga la página
