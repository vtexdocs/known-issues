## Sumario

Al crear un carrito a partir de un pedido anterior (mediante el botón «Volver a pedir» de la pantalla «Mis pedidos»), no se conservarán los accesorios ni los servicios del pedido original.

## Simulación

1. Realice un pedido de un producto con accesorios y/o servicios;
2. Acceda a la pantalla «Mis pedidos»;
3. Haga clic en «Volver a pedir»;
4. Observe que el carrito que se ha generado solo incluye los SKU del pedido anterior.

## Workaround

Actualmente, no existe ninguna solución alternativa para este caso.
