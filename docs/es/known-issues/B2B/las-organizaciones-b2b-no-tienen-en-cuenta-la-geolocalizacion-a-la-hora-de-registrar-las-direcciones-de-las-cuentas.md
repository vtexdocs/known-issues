## Sumario

La organización B2B no dispone de ninguna opción para configurar una dirección cuando las cuentas utilizan la geolocalización, lo que provoca que los artículos no estén disponibles.

## Simulación

- Configura una cuenta para que utilice la geolocalización al finalizar la compra;
- Asegúrate de que se trata de una cuenta de países que utilizan opciones predefinidas, por ejemplo, Argentina, Chile, Colombia, etc.;
- Crea una organización;
- Intenta comprar cualquier producto; siempre estará indisponible.

## Workaround

Cambia la configuración para utilizar únicamente el código postal.
