## Sumario

Cuando un usuario tiene muchas organizaciones registradas y utiliza la ventana modal para cambiar de una a otra, algunas de ellas aparecen duplicadas en la lista.

## Simulación

- Configura varias organizaciones para la misma dirección de correo electrónico;
- Inicia sesión y abre la ventana modal para cambiar de organización;
- Algunas organizaciones aparecerán duplicadas en la lista.
