## Sumario

En ocasiones, el XML puede devolver un error 500; esto suele ocurrir principalmente con archivos XML de gran tamaño.

## Simulación

Intenta cargar un archivo XML de gran tamaño.

## Workaround

provisional**

- Intenta volver a cargar el XML; esto suele resolver el problema;
- Utiliza más de un archivo XML.
