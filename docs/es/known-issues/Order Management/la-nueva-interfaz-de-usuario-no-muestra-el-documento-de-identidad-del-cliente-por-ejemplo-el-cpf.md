## Sumario


La Nueva UI no muestra el documento de identificación del cliente en los pedidos. Por ejemplo como CPF, ocurre porque la política comercial de cuentas tiene configurado otro país.


#### Simulación


Puede configurar un país diferente en la política de comercio y hacer una compra utilizando esta configuración de política en la tienda de pago. Proceso normal.
Después de esto abra el admin y consulte el pedido correctamente, en la información del cliente no verá el ID del documento del cliente.

## Workaround


Por ahora no hay solución.
