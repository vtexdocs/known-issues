## Sumario

Cuando un usuario cierra una solicitud de suscripción, el área «Mis suscripciones» muestra la dirección de entrega; sin embargo, cuando esto se hace con un punto de recogida, la interfaz de usuario no puede establecer la asociación y muestra el siguiente mensaje de error: «Dirección no válida o inexistente»; no obstante, el proceso de firma se lleva a cabo con normalidad;

## Simulación

- Crea una suscripción con la opción de recogida en un punto de recogida;
- Accede al área «Mis suscripciones»;
- En esta área verás una advertencia en la interfaz de usuario que indica que hay un error en el registro de tu dirección;

## Workaround

No hay ninguna solución alternativa para este caso;
